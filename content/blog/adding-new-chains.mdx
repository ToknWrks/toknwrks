to add a new chain, you need to update several parts of your application. Here's a step-by-step guide on what needs to be updated:

Update chains.ts: Add the new chain configuration.

Update WalletContext.tsx: Add state and functions for the new chain.

Update WalletConnection.tsx: Handle connection logic for the new chain.

Create dashboard-newchain.tsx: Create a new dashboard component for the new chain.

Update page.tsx: Include the new dashboard component and handle its state.
